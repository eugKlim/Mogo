# Что реализовано в этом проекте?

1. При отсутствии контента, footer не прилипает к header.

2. Переиспользуемый аккордион.

3. Слайдер, библиотека `splide slider`.

4. Адаптация под малые устройства.

5. Библиотека `React-modal`, попап. Если будет много контента, то стоит максимальная высота, и появится скролл, так же его цвет, меняется исходя из выбранного режима dark/light.

6. Добавлен google maps.

7. Dark mode(с записью в локальное хранилище).

8. Burger menu.

9. Фиксация header при скроле.

10. Роутинг по навигацие.
    - Переходя на страницу, если контента меньше чем экран по высоте(нету скрола в низ), то скролл пропадать не будет.
    - Пока страница подгружается, отображается спинер.
    - Так же добавил ленивую загрузку компонентов.

11. Определение форматов, проверяем, поддерживает ли браузер форматы `.avif` и `.webp`. Для фото библиотека `react-lazy-load-image-component` он же `lazyloading` вместе с тегами `picture`, если браузер поддерживает `.avif` то все фото будут подгружатся в этом формате, если не поддерживает `.avif` тогда попробует `.webp` если и его не поддерживает, то вернет исходный формат. Маленькие иконки тоже самое, только для них lazyloading убрал, тк как для телефонов будет так легче. `Svg` форматы в lazyloading тоже не включал. Для фонов написан скрипт, который при заходе на сайт, проверяет форматы, так же как и для фото, но все фоны загружаем сразу без ленивой загрузки, чтобы при скроле не лагало. Ну и соответственно все фото `.png` и `.jpg` проебразованы в `.avif` и `.webp` ну и исходные изображение тоже имеются.
 
12. Мета теги для сео, через библиотеку `react-helmet-async`.

13. На телефонах / планшетах, hover блоки, будут вызваны при нажатии на кнопку.

14. Теги для скрин ридеров и семантическая верстка ( accessibility ).

16. В код css добавлены префиксы для поддержки старых браузеров ( autoprefixer ).

17. Для иконок использован формат `.svg`, в отдельных файлах `.jsx`, чтобы можно было при навидении, изменять иконку. и в коде используются технологии grid layout & flex box.

18. В коде использую так же технологии `grid-layout` и `flex-box`.

## Как установить проект себе?
1. Клонировать репозиторий:

```ruby
git clone https://github.com/eugKlim/Mogo.git
```

2. Установить зависимости:

```ruby
npm install
```

3. Запустить проект:

```ruby
npm run dev
```
